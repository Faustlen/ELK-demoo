### Практическое задание по работе с системой логирования

1) Запустить стек логирования, используйте docker-compose
2) Через веб-интерфейс подключиться к поисковому движку (первоначальная настройка)
3) Сгенерировать тестовые логи
4) Настроить logstash.conf для передачи логов в систему
5) Изучить веб-интерфейс, выполнить поиск ранее записанных логов
